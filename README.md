# Motion-GPT

## Introduction

Project homepage of [Motion-GPT](https://motion-gpt.github.io/).

## Usage

To install and run Motion-GPT, follow these steps:

1. Clone the repository
```bash
git clone https://github.com/Motion-GPT/motion-gpt.github.io.git
cd motion-gpt.github.io
```
2. Install dependencies
```bash
npm install
```
3. **Comment the asset prefix for local debugging**
https://github.com/Motion-GPT/motion-gpt.github.io/blob/b8c5cd07bd225b258de0aadd71ff53b23432b693/next.config.mjs#L9

4. Start the project
```bash
npm start
```

Once the application is running, visit [http://localhost:3000](http://localhost:3000) in your browser to view the page.
